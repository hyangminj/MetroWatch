## 🚀 출시 단계

### 1단계: Google Play Console 준비
1. **개발자 계정 생성** ($25 1회 결제)
   - https://play.google.com/console
   - Google 계정으로 로그인
   - 개발자 정보 입력

### 2단계: 앱 생성
1. "모든 앱" → "앱 만들기"
2. 앱 이름: MetroWatch
3. 기본 언어: 한국어
4. 앱 유형: 앱
5. 무료/유료: 무료

### 3단계: 스토어 등록정보 입력
1. **앱 세부정보**
   - 앱 이름
   - 짧은 설명
   - 전체 설명
   - 앱 아이콘
   - Feature Graphic (선택)
   - 스크린샷

2. **분류**
   - 앱 카테고리: 음악 및 오디오
   - 태그 추가

3. **연락처 세부정보**
   - 이메일 주소
   - 웹사이트 (GitHub 링크)

### 4단계: AAB 업로드
1. **프로덕션** → **새 버전 만들기**
2. AAB 파일 업로드 (`app/release/app-release.aab`)
3. 버전 이름: 1.0
4. 출시 노트 작성

### 5단계: 콘텐츠 설정
1. 앱 액세스 권한
2. 광고 여부
3. 콘텐츠 등급
4. 타겟 고객
5. 데이터 보안

### 6단계: 검토 및 출시
1. 모든 항목 확인
2. "검토" 클릭
3. 문제 없으면 "프로덕션으로 출시"
4. 심사 대기 (보통 며칠~1주일)
