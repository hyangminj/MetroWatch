# MetroWatch - 스크린샷 가이드

## Play Store 요구사항

### Wear OS 앱 스크린샷
- **최소 개수**: 1개 (권장: 3-5개)
- **형식**: JPEG 또는 24-bit PNG (알파 채널 없음)
- **크기**: 최소 320px, 최대 3840px
- **권장 해상도**: 384x384 (Wear OS Small Round), 450x450 (Large Round)

---

## 촬영할 스크린샷

### 1. **메인 화면** (필수)
**촬영 방법:**
1. Wear OS 에뮬레이터에서 앱 실행
2. BPM: 120, 박자: 4/4 상태
3. START 버튼 누르기 전 화면
4. Android Studio: Run → Screenshot (또는 Ctrl/Cmd + S)

**보여줄 내용:**
- 120 BPM 표시
- 4/4 박자 표시
- BPM 조절 버튼 (-5, -1, +1, +5)
- 녹색 START 버튼

---

### 2. **실행 중 화면** (필수)
**촬영 방법:**
1. START 버튼 클릭
2. 비트 카운터 "1/4" 또는 "2/4" 표시될 때 스크린샷

**보여줄 내용:**
- 녹색 120 BPM (실행 중 표시)
- 비트 카운터 (예: 2/4)
- 빨간색 STOP 버튼

---

### 3. **볼륨 조절 화면** (권장)
**촬영 방법:**
1. 화면을 아래로 스크롤
2. "소리: 80%" 및 "진동: 80%" 부분이 보이도록

**보여줄 내용:**
- 소리 볼륨 조절 (80%)
- 진동 강도 조절 (80%)
- +/- 버튼

---

### 4. **다른 박자** (선택)
**촬영 방법:**
1. 박자 "4/4"를 탭하여 "3/4" 또는 "6/8"로 변경
2. 스크린샷

**보여줄 내용:**
- 다른 박자 설정 (예: 3/4)
- "박자 (탭해서 변경)" 안내 문구

---

## 스크린샷 촬영 방법

### Android Studio에서 촬영
1. Wear OS 에뮬레이터 실행
2. 앱 실행
3. **Camera 아이콘 클릭** (에뮬레이터 오른쪽 툴바)
   - 또는 `Ctrl + S` (Windows/Linux)
   - 또는 `Cmd + S` (Mac)
4. 저장 위치: 자동으로 지정된 폴더에 저장됨

### 파일 위치
스크린샷은 보통 다음 위치에 저장됩니다:
```
~/Desktop/
또는
~/Pictures/
```

## 스크린샷 편집 (선택)

### 추천 도구
- **macOS**: Preview, Pixelmator
- **Windows**: Paint, GIMP
- **온라인**: Figma, Canva

### 편집 팁
1. **배경 추가** (선택)
   - 워치 이미지 프레임 추가
   - 배경색: 검은색 또는 그라데이션

2. **텍스트 추가** (선택)
   - "정확한 박자"
   - "진동 + 소리"
   - "다양한 박자 지원"

3. **크기 최적화**
   - 384x384 (Small Round)
   - 450x450 (Large Round)

## 스크린샷 체크리스트

촬영 전 확인사항:
- [ ] 앱이 최신 버전인지 확인
- [ ] 화면이 깔끔한지 (시간 표시 확인)
- [ ] 글씨가 선명한지
- [ ] 주요 기능이 잘 보이는지

촬영 후 확인사항:
- [ ] 파일 형식: PNG 또는 JPEG
- [ ] 파일 크기: 최소 320px
- [ ] 화질 선명
- [ ] 최소 1개 이상 (권장 3-5개)

---

## 빠른 촬영 가이드

**5분 안에 필수 스크린샷 촬영:**

1. 에뮬레이터 실행
2. 앱 실행
3. 스크린샷 #1: 메인 화면 (START 버튼 보이게)
4. START 클릭
5. 스크린샷 #2: 실행 중 (비트 카운터 보이게)
6. 완료!

나머지 스크린샷은 나중에 추가 가능합니다.
